# e-paperback
 An application for users to purchase a book of their choosing and save the book in their personal libraries
 
 ## GETTING STARTED
 Required JAVA 20. Download: https://www.oracle.com/java/technologies/javase/jdk20-archive-downloads.html
 Required Maven 3.9.1. Download: https://maven.apache.org/download.cgi
 
 1. Install Node.js and Angular CLI: 
    npm install -g @angular/cli
 
 2. Run spring boot
    ./mvnw spring-boot:run
    
 3. Run angular 
    ng serve
 
 4. Access the Angular application by navigating to http://localhost:<port number> in your web browser.
